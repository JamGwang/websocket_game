# 웹소켓 게임 만들기
<br>

### 패킷 구조

| 필드 명 | 타입 | 설명 |
|---|---|---|
| handlerID | int | 요청을 처리할 서버 핸들러의 ID |
| userId | int | 요청을 보내는 유저의 ID
| clientVersion | string | 현재 클라이언트의 버전 |
| payload | JSON | 요청 내용  |


---
<br>

+ 스테이지 마다 별도의 획득 점수
+ 스테이지 점수에 따라 아이템 생성
+ 아이템 획득 시 해당 아이템에 지정된 점수 획득

<br>

### 해결해야하는 과제
* 점수 검증 
점수 검증 시 아이템 획득 점수를 포함한 점수 합산 및 검증 로직 미완성<br>
